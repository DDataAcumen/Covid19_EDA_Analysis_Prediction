# Covid19 Pandemic_EDA-Analysis-and-Predictions

This Git Repository is designed  to Understand the impact of COVID-19 across Globe. It provides strong visulizations that  help us perceive how this Global Epidemic started and in no time flourished havoc around the world. Further Machine Learning Modelling applied gives us an idea of how the number of active cases cross economies is expected to rise, and how the number of fatalities will act with respect to number of cases and time ofcourse.
